# DonorBankBd

A professional blood donation management platform for Bangladesh. This platform connects blood donors with those in need, providing a seamless and efficient way to manage life-saving requests.

## Features

- **Donor Registration**: Easy sign-up for volunteers.
- **Blood Search**: Find donors by blood group and location across 64 districts.
- **Emergency Requests**: Submit urgent blood requests for immediate matching.
- **Admin Dashboard**: Secure portal for managing donors, requests, and emergency contacts.
- **Real-time Stats**: Live tracking of active donors and successful donations.

## Tech Stack

- **Frontend**: React 19, TypeScript, Vite
- **Styling**: Tailwind CSS 4
- **Icons**: Lucide React
- **Backend**: Supabase (PostgreSQL + Auth)
- **Deployment**: Vercel / Cloud Run

## Getting Started

1. Clone the repository.
2. Install dependencies: `npm install`
3. Set up environment variables in `.env`.
4. Run the development server: `npm run dev`
